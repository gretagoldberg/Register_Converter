ATLAS ITk Strips ASICs register converters

ASICs:
ABCStar v1
HCCStar v1
AMACStar

Convert between output signal (register content) and readout information for each register

Input:
Desired register by name (exact wording necessary), decimal number, or hexadesimal number
Signal in hexadecimal, binary, or decimal
OR
Desired information as appropriate

Output:
Corresponding information
OR
Signal in hexadecimal, binary, or decimal
Information in text form
File with information
